## ace-step dataset toolkit

A toolkit to create proper json dataset file for lora training in ace-step-1.5.
We utilize ace-step-captioner to generate captions, genius-api to download lyrics and mixxx to generate the proper bpm and key metadata.

**Pre-requisits**
- Only tested on Windows
- UV installed

**Installation**
- Start install.bat and wait for the dependencies to download
- Use run_toolkit.bat to run the CLI
